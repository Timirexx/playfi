# PLAYFI 🎮 Hedera Arcade

**PLAYFI** is a premium, high-performance gaming platform built on the **Hedera Network**. It focuses on provably fair mechanics, seamless wallet integration, and high-transaction-volume gameplay.

## 🚀 Live Demo
Experience the platform live: [https://playfi-liart.vercel.app/](https://playfi-liart.vercel.app/)

## 🎮 Featured Games

### 💣 Mines Field
A high-stakes game of strategy and probability.
- **Tuning**: High-density bomb placement (15 bombs) for a 20% success profile.
- **Fairness**: Board generation is finalized *before* the first click, ensuring complete transparency.
- **Multipliers**: Survival-based rewards with combinatorial probability math.

### 🎡 Spin-to-Win
A professional-grade wheel game with weighted outcomes.
- **80/20 Distribution**: Mathematically balanced with 8 losses and 2 win segments per 10 slices.
- **Physics**: Smooth 4-second spin animation with variable deceleration.
- **Rewards**: Small (1.5x) and Big (5x) wins.

### 📈 Crash (Coming Soon)
A classic multiplier game based on a fair probability curve.

## 🔗 Hedera Integration
- **Wallet Support**: Comprehensive support for **Hashpack**, **MetaMask**, **Keplr**, and **Phantom**.
- **On-Chain Betting**: Real-time HBAR transaction handling with secure debouncing.
- **Environment**: Compatible with Hedera Mainnet (Chain ID 295) and Testnet (Chain ID 296).

## 🛠️ Technology Stack
- **Frontend**: Vanilla HTML5, CSS3, and JavaScript (ES6+).
- **Styling**: Custom "Glassmorphism" UI with deep blue and black aesthetics.
- **Deployment**: Optimized for Vercel.

## 🛡️ Fairness
Every game uses **Provably Fair** mathematical models (5% House Edge) to ensure a balanced economy and long-term sustainability.
